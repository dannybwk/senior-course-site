開機與解鎖

為什麼重要？因為這是使用手機的第一步！

  1. 長按電源鍵 3 秒，直到螢幕亮起
  2. 輸入密碼 / 圖形 / 使用臉部辨識
  3. 成功進入手機主畫面

調整音量

音量太大或太小，都會影響使用體驗

  • 用手機側邊的音量鍵調整
  • 若沒反應，檢查靜音鍵（iPhone）或設定

調整亮度

螢幕太暗或太亮，容易看不清楚或耗電

  1. 打開「設定」或「快速選單」
  2. 找到亮度控制條
  3. 上下滑動調整亮度

連接 Wi-Fi

上網更快、省流量

  1. 開啟「設定」→ Wi-Fi
  2. 選擇家裡或咖啡廳的網路名稱
  3. 輸入密碼 → 按「連線」

常見問題：App找不到

找不到LINE或相機App怎麼辦？

  • 在主畫面往下滑 → 輸入App名稱搜尋
  • 確認是否被移到資料夾內
  • 從App Store / Play商店重新下載

常見問題：手機卡住

如果畫面不動了

  • 長按電源鍵直到出現「重新啟動」
  • 若無法，長按電源+音量鍵強制重啟
