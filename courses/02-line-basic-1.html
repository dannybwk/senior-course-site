如何加入好友：總覽

  • 方式一：使用 QR Code 掃描
  • 方式二：搜尋 LINE ID
  • 方式三：手機號碼加入
  • 方式四：從群組邀請中加入

加入好友：使用 QR Code

  • 點選主頁 → 加號 → 加好友
  • 選擇「行動條碼」
  • 掃描對方提供的 QR Code

產生自己的 QR Code

  • 可請朋友掃描你的 QR Code 加好友
  • 點選主頁 → 加好友 → 我的條碼

加入好友：使用 LINE ID

  • 輸入對方的 LINE ID（必須對方已開啟公開）
  • 點選搜尋 → 輸入 ID → 送出

加入好友：使用手機號碼

  • 輸入對方的手機號碼來查找
  • 若對方帳號綁定了該號碼，會顯示出來

從群組邀請加好友

  • 進入同一個群組後，可直接互加
  • 點開成員列表 → 加為好友

好友邀請提醒與回應

  • 當有人加你好友，會出現通知
  • 可以選擇接受或封鎖

加入好友後的第一步

  • 建議互傳貼圖，測試雙方是否已加成功
  • 可留言「你好」、「我是○○老師」等訊息

好友名單管理

  • 可長按好友 → 編輯名稱
  • 分類好友：家庭、朋友、學員...

練習時間：加入好友任務

  1. 與左右鄰居交換 LINE ID 或 QR Code
  2. 實際嘗試加入對方為好友
  3. 傳出一則貼圖，確認成功

好友設定：封鎖與解除封鎖

  • 長按好友 → 選擇封鎖
  • 封鎖後對方無法傳訊給你
  • 解除封鎖方式：設定 → 好友 → 封鎖名單

好友設定：隱藏與刪除

  • 隱藏好友：從主頁不顯示，但資料仍在
  • 刪除好友：需先封鎖 → 再進入封鎖名單刪除

好友提醒：LINE ID 不可變更

  • LINE ID 一旦設定無法更改
  • 若忘記，可分享 QR Code 給對方

常見問題：加好友失敗怎麼辦？

  • 確認對方 LINE ID 正確拼寫
  • 對方有開啟「允許被搜尋」嗎？
  • 手機是否連上網路？

小技巧：分享好友給他人

  • 長按好友 → 選「分享」
  • 可將此人資訊傳給群組或其他朋友
